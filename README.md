# Morrison Solutions

A professional website for Morrison Solutions - Innovative Consulting & Professional Services.

## Features

- **Responsive Design**: Mobile-friendly website that works on all devices
- **Modern UI**: Clean and professional interface with smooth animations
- **Multiple Sections**:
  - Hero section with call-to-action
  - Services showcase (Business Consulting, Digital Solutions, Data Analytics, Technical Support)
  - About section with company statistics
  - Contact form for inquiries
  - Sticky navigation bar

## Files

- `index.html` - Main HTML file
- `styles.css` - Styling and animations
- `script.js` - Interactive features and form handling

## Deployment

This website is deployed using GitHub Pages. To access it, visit:
```
https://morrisonc11998-eng.github.io/morrison-solutions
```

## Customization

You can customize the following:

### Content
- Edit the service descriptions in the Services section
- Update company statistics in the About section
- Modify contact information in the footer

### Colors
- The primary gradient color is `#667eea` to `#764ba2`
- Edit these values in `styles.css` to change the theme

### Services
- Add or remove service cards in the Services section
- Update icons and descriptions as needed

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- GitHub Pages
